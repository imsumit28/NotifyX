How to integrate NotifyX

Three steps to add real-time notifications to any application — free, no credit card required.

1

Get an API Key

Request a free API key. Your key starts with nx_ and authenticates all requests from your backend.

2

Send from your server

Call POST /api/notify with your API key. Specify recipient, type, and payload. That's it — NotifyX handles queueing, retries, and delivery.

3

Receive in real-time

Connect your frontend via Socket.io. Users receive notifications instantly — even if they were offline, they get synced on reconnect.
